Installation:

1. Visit chrome://extensions 2. Check the developer mode checkbox 3. Download the extension (.zip) 4. Press "Load unpacked" and upload the folder you downloaded
(or an equivalent for your browser)
This allows you to install an extension manually
and then unzip the file on your device and extract the files as a folder
The button should be on the top left of the chrome://extensions page.
